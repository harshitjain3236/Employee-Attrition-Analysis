# Employee Attrition Analysis

## Overview
This project predicts whether an employee is likely to leave the company using Machine Learning.

## Dataset
IBM HR Analytics Employee Attrition Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Accuracy
88.10%

## Workflow
- Data Cleaning
- Label Encoding
- Train-Test Split
- Model Training
- Prediction
- Model Evaluation
- Feature Importance Analysis

## Conclusion
The Random Forest model achieved approximately 88.10% accuracy in predicting employee attrition.
